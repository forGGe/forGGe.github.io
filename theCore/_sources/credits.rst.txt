Invoking Sphinx from CMake
~~~~~~~~~~~~~~~~~~~~~~~~~~

`Sphinx and CMake: Beautiful Documentation For C++ Projects`_ by Eric Scott Barr
is a great article, describes how to integrate those tools together.
This is how theCore builds and deploys its documentation.

.. _`Sphinx and CMake: Beautiful Documentation For C++ Projects`: https://eb2.co/blog/2012/03/sphinx-and-cmake-beautiful-documentation-for-c-projects/
